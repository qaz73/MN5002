# MN5002 Advanced NLP Exercise Repository

## Description
This repository is for the use of MN5002 Advanced NLP so that students can present their work and comment on each other's contributions.

## Usage
After you have provided the module leader with your GitHub ID, you can submit your works in progress to the respective folders (Section 1, Section 2 and Section 3) on your own branches of the repo. After which you should mention you have done so on the Discussion Forum along with a permalink to the file you have provided. In this way, people can review each other's code, learn from them and discuss them.

## Support
If you have any difficulties setting up on this repository, please contact the course module leader.

## Contributing
Your code contributions are welcome and corresponding discussion of the work is encourage through the discussion forums on Brightspace. When you make a submission on GitHub, please highlight it in the appropriate discussion section.

Please note that while GitHub is for sharing of work in progress, final submissions of homeworks must be through Brightspace itself (by submitting the python notebook to the relative Assignment).
